Ractive-event-viewport

Event that is fired as soon as an element appears in the userâ€™s viewport.It uses getBoundingClientRect() method.

Requires: RactiveJS

Usage

The script adds viewport event to RactiveJS. The event will only fire when the element comes in to view of the viewport, and out of viewport. It won't keep triggering element remains in view or not in view.

<div on-viewport="someFunc">Lorem Ipsum...</div>

ractiveObj.on('someFunc', function(event){
   event.original.current // Current state
   event.original.previous // Previous state
   event.original.visible // same as event.original.current
});

How it works

An interval in the background checks for the position of the bounding rectangle of element against the viewport dimensions and the parent's scroll position. If it detects any change in view state, it triggers an event on element.

Browser Compatibility

Browsers that support getBoundingClientRect().
